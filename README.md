# Happy Birthday Fahimah! 🎂💝

An interactive 3D birthday book with beautiful page-turning animations.

## 📖 About This Project

This is a digital birthday book created with love for Fahimah. It features:
- A realistic 3D book that opens and closes
- Beautiful page-turning animations
- Memory pages with photos
- "Best Eats" top 3 rankings
- A heartfelt final message

## 🚀 Live Demo

Visit the live book at: `https://[your-username].github.io/[repository-name]/`

## 📸 Adding Your Photos

Add these photos to the `images` folder:

| File Name | What It Should Be |
|-----------|-------------------|
| `us.jpg` | A photo of you two together (for the cover) |
| `memory1.jpg` | Your first special memory |
| `memory2.jpg` | Your second special memory |
| `memory3.jpg` | Your third special memory |
| `memory4.jpg` | Your fourth special memory |
| `food1.jpg` | Your #1 favorite restaurant/meal |
| `food2.jpg` | Your #2 favorite spot |
| `food3.jpg` | Your #3 favorite food memory |

**Note:** If you don't add photos yet, the book will show beautiful placeholder images automatically!

## 🎮 How to Use

- **Click "Open Book"** to start
- Use the **Next/Back buttons** to turn pages
- Use **keyboard arrow keys** (← →) to navigate
- Click on any page for a zoom effect

## 🛠️ Files

- `index.html` - The entire book (HTML, CSS, and JavaScript all in one)
- `images/` - Folder containing your photos
- `README.md` - This file

## 💕 Customize the Messages

Open `index.html` and search for these sections to personalize:

- Memory descriptions (search for "When It All Began")
- Food descriptions (search for "Best Eats")
- Final message (search for "Here's to Many More")

## 📱 Features

- Works on phones, tablets, and computers
- Realistic 3D page turns
- Beautiful gradient backgrounds
- Smooth animations
- No installation needed - works in any browser
